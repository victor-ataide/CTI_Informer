🚀 INÍCIO RÁPIDO (5 MINUTOS):
═════════════════════════════════════════════════════════════════════════════

1. INSTALAR OLLAMA (em novo terminal):
   curl https://ollama.ai/install.sh | sh
   ollama pull mistral

2. CLONAR/ENTRAR NO DIRETÓRIO:
   cd ~/Desktop/CTI

3. EXECUTAR SETUP:
   bash setup.sh

4. CONFIGURAR DISCORD:
   nano config.json
   # Procure por: "webhook_url": "SEU_LINK_AQUI"
   # No Discord: Servidor → Integrações → Webhooks

5. RODAR SISTEMA:
   # Terminal 1:
   ollama serve
   
   # Terminal 2:
   source venv/bin/activate
   python main.py --daemon

📊 O QUE O SISTEMA FAZ:
═════════════════════════════════════════════════════════════════════════════

1️⃣  COLETA (30 segundos)
   └─ 15+ fontes RSS de segurança → ~50 artigos

2️⃣  PROCESSAMENTO (15 segundos)
   └─ Remove HTML, normaliza texto → Artigos limpos

3️⃣  ANÁLISE (10-30 segundos por artigo)
   └─ LLM extrai APTs, malware, setores, TTPs

4️⃣  EXTRAÇÃO IOCs (5 segundos)
   └─ IPs, domínios, URLs, emails, hashes, CVEs

5️⃣  CLASSIFICAÇÃO (1 segundo)
   └─ Verifica: crítico? financeiro? infraestrutura?

6️⃣  ALERTAS (1 segundo)
   └─ Se crítico/financeiro/infra → Discord!

7️⃣  ARMAZENAMENTO (1 segundo)
   └─ JSON em data/results/
