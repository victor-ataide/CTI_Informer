🔷 MÓDULOS CORE (6 arquivos Python - ~2000 linhas de código):

   1. 📥 collector.py (296 linhas)
      └─ Coleta de múltiplas fontes RSS/blogs de segurança
      └─ 15+ fontes pré-configuradas
      └─ Adicionar/remover fontes facilmente

   2. 🔧 parser.py (272 linhas)
      └─ Processamento e limpeza de HTML
      └─ Normalização de texto
      └─ Detecção de setores e severidade

   3. 🎯 ioc_extractor.py (384 linhas)
      └─ Extração de Indicadores de Comprometimento (IoCs)
      └─ IPs (v4/v6), domínios, URLs, emails
      └─ Hashes (MD5/SHA1/SHA256/SHA512)
      └─ CVEs, registry keys, mutexes
      └─ ~10 tipos diferentes de IoCs

   4. 🧠 intel_engine.py (381 linhas)
      └─ Análise com LLM local (Ollama)
      └─ Extração de APTs, malware, setores
      └─ Mapeamento MITRE ATT&CK
      └─ Classificação automática

   5. 📱 notifier.py (318 linhas)
      └─ Alertas formatados para Discord
      └─ Embeds coloridos por severidade
      └─ Todas as informações técnicas
      └─ Batch processing

   6. 🎮 main.py (412 linhas)
      └─ Orquestrador completo do sistema
      └─ 5 fases: coleta → processamento → análise → alertas → armazenamento
      └─ Modo daemon (execução contínua)
      └─ Logging estruturado

                            🚀 COMO COMEÇAR
═══════════════════════════════════════════════════════════════════════════════

1️⃣  INSTALAÇÃO RÁPIDA (5 minutos):

    # 1. Instalar Ollama
    curl https://ollama.ai/install.sh | sh
    ollama pull mistral

    # 2. Ir para diretório
    cd ~/Desktop/CTI

    # 3. Rodar setup automático
    bash setup.sh

    # 4. Configurar Discord
    nano config.json
    # Procure por: "webhook_url" e adicione seu link

    # 5. Rodar sistema
    ollama serve              (Terminal 1)
    source venv/bin/activate  (Terminal 2)
    python main.py --daemon

2️⃣  PRIMEIRO USO:

    # Terminal 1: Inicie Ollama
    ollama serve

    # Terminal 2: Execute sistema UMA VEZ
    cd ~/Desktop/CTI
    source venv/bin/activate
    python main.py

    # Será coletado ~50 artigos
    # Analisará com LLM
    # Se encontrar ameaças críticas/financeiro/infra → Envia Discord!
    # Salva tudo em data/results/

3️⃣  RODAR 24/7:

    python main.py --daemon   # Executa indefinidamente a cada 1 hora

4️⃣  TESTAR:

    python main.py --test     # Envia teste para Discord
